## Why Antimatter?

Developing with autonomous AI agents often means leaving them running for hours. Antimatter ensures you aren't chained to your desk — **monitor, control, and interact with your agent from anywhere** using your phone, secured by enterprise-grade cryptography.

<div class="features-grid">
  <div class="feature-card">
    <span class="feature-icon">📡</span>
    <h3>Real-Time Streaming</h3>
    <p>Watch your agent's thought process, tool calls, and file edits character-by-character as they happen.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">🔐</span>
    <h3>Zero Trust Security</h3>
    <p>256-bit pairing tokens + Ed25519 handshakes + Cloudflare tunnels. No open ports. No static IPs.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">📸</span>
    <h3>QR Pairing</h3>
    <p>One scan transfers the WebSocket URL, token, and Ed25519 public key. Cryptographically paired in seconds.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">🗂️</span>
    <h3>Workspace Browser</h3>
    <p>Browse your IDE workspace files in real-time and view or edit any file directly from your phone.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">💬</span>
    <h3>Remote Prompting</h3>
    <p>Send multimodal prompts (text + images) directly to your AI agent and start new conversations on the go.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">🖥️</span>
    <h3>Native PTY Terminal</h3>
    <p>Full interactive shell with ANSI rendering. Supports vim, htop, ssh. Powered by Termux (Android) & SwiftTerm (iOS).</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">🔒</span>
    <h3>Biometric Auth Gate</h3>
    <p>Fingerprint or Face Unlock required before the app establishes any connection to your agent.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">🔍</span>
    <h3>Full-Text Search</h3>
    <p>FTS across your entire local SQLite trajectory database — find past commands and agent responses instantly.</p>
  </div>
  <div class="feature-card">
    <span class="feature-icon">📲</span>
    <h3>Push Notifications</h3>
    <p>Get notified when your agent completes a task, needs approval, or encounters an error — even when backgrounded.</p>
  </div>
</div>

## Supported AI Agents

Antimatter's **Independent Adapter Model** means it works with any AI agent — not just one.

| Adapter | Agent | Technology | Status |
|---|---|---|---|
| `ag` | Google Antigravity IDE | TypeScript · VS Code Extension | <span class="badge badge-shipped">✓ Shipped</span> |
| `ag2` | Google Antigravity 2.0 | Python daemon | <span class="badge badge-shipped">✓ Shipped</span> |
| `cc` | Claude Code (Anthropic) | Node.js · Claude SDK | <span class="badge badge-shipped">✓ Shipped</span> |
| custom | Any Agent | Any Language (WebSocket IPC) | <span class="badge badge-planned">Open</span> |
